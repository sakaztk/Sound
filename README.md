# Sound

```
>Sound -h
Usage:
Sound [-g | -im | [[-m | -u] [VOLUME]]

Options:
  -g,  --get     Get the current volume.
  -im, --ismute  Get the mute status.
  -m,  --mute    Mute the volume.
  -u,  --unmute  Unmute the volume.
  VOLUME         Set specifies the volume.

Examples:
    > Sound -m
    > Sound -u 50
    > Sound 10
    > Sound +10
    > Sound -10
```
